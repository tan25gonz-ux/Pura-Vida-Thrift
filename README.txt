🚀 Pura Vida Thrift – Modo de Prueba Local + Firebase Hosting Ready

1️⃣ Cómo probar localmente:
- Abre la carpeta "puravidathrift" en Visual Studio Code.
- Instala la extensión "Live Server".
- Clic derecho en "local-test.html" → "Open with Live Server".
- Inicia sesión con tu cuenta (tan25gonz@gmail.com).
- Ya puedes agregar, editar y eliminar productos.
- Abre "public/index.html" para ver la tienda actualizada.

2️⃣ Cómo importar los productos de ejemplo a Firestore:
- Ve a https://console.firebase.google.com > Firestore Database.
- Crea la colección llamada **productos**.
- Usa los datos del archivo productos-ejemplo.json como ejemplo.

3️⃣ Para subir tu sitio a Firebase Hosting:
- Abre la terminal en la carpeta principal y ejecuta:
    firebase login
    firebase deploy

4️⃣ Tu sitio estará activo en:
   https://puravidathrift.web.app
